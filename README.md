# 🚀 GenX FX Dashboard

Production-ready trading monitor for the **GenX_FX AI Trading System**. This dashboard provides real-time visualization of AI-generated signals, system logs, and account performance.

## ✨ Key Features
- **Real-time Signal Terminal**: Live BUY/SELL alerts with entry, TP, SL, and confidence metrics.
- **AI Engine Monitoring**: Monitor model health, inference latency, and accuracy in real-time.
- **System Terminal**: Live log stream from the core trading engine.
- **Performance Analytics**: Interactive charts for profit tracking and account growth.
- **Secure Access**: Integrated Google Authentication with role-based permissions.

## 🛠️ Tech Stack
- **Frontend**: React, TypeScript, Tailwind CSS, Framer Motion.
- **Backend**: Firebase (Auth & Firestore).
- **Charts**: Recharts.
- **Icons**: Lucide React.

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- Firebase Project

### Setup
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Configure environment variables in `.env`:
   ```env
   GEMINI_API_KEY=your_key
   JULES_API_KEY=your_key
   JULES_API_TOKEN=your_token
   DROPBOX_API=your_key
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## 🛡️ Security
This application uses Firestore Security Rules to ensure data isolation. Only authorized users can access sensitive trading signals and logs.
